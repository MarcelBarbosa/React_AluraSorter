## Descriçao:
Projeto utilizando React e blibliotecas de testes para a criação e testes de uma página de cadastro e sorteios de Amigos Secreto

## Principais Ferramentas:
* React
* Typescript
* Recoil
* React-router-dom
* React-testing-library
* Hooks
* Jest
* Jest-dom
* TDD
